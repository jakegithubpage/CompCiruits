# CompCiruits
CompCiruits - an application made for EE students. The goal of this application is to put gamification and a competitive edge into circuit analysis. It uses vector-based algorithms to create randomly generated circuits of desired difficulty and genre that users are timed and measured on. Results from each problem are ranked on a live leaderboard!
